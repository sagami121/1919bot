# 1919bot
毎日19時19分に自動で「1919」と送信するDiscord bot

# セットアップ
1.リポジトリをクローン
```
git clone https://github.com/sagami121/1919bot.git
cd 1919bot
```
2.必要なライブラリをインストール
```
pip install -r requirements.txt
```
3.プロジェクト直下に .env を作成して以下を記入：
```
DISCORD_TOKEN=Botトークン
CHANNEL_ID=チャンネルID
```

# 起動方法
```bash
python main.py
```
